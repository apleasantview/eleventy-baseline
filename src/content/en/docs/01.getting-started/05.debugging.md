## What it does
- Adds a Nunjucks global `_navigator` to inspect the full render context (handy for debugging).
- When `enableVirtualTemplate` is true, registers a virtual page from at `/navigator-core.html`.

### Defaults
- No custom options.
- Virtual page only when `verbose` is enabled in Baseline options.

### How it works
- Registers `_navigator()` to return the current Nunjucks context.
- Registers `_context()` â€” a childe of `_navigator()`
- If `ELEVENTY_ENV !== "production`, ads the bundled `core-navigator.html` template as a page with a fixed permalink.

### Tips
- Turn on `verbose: true` in your Baseline plugin options to get the debug page.
- `_navigator()` is useful inside templates to inspect the shape of Eleventy data while developing.
